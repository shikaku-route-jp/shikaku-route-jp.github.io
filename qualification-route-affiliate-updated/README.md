# 資格ルート診断 本格公開用パッケージ

## 推奨構成
- GitHubでこのフォルダを管理
- Cloudflare Pages または Netlify にGit連携でデプロイ
- 独自ドメインを接続
- HTTPSを有効化
- _headers によるセキュリティヘッダー適用

## ファイル
- index.html: サイト本体
- _headers: Cloudflare Pages / Netlify向けセキュリティヘッダー
- robots.txt: 検索エンジン向け設定

## 公開後にやること
1. robots.txt の Sitemap URL を独自ドメインに変更
2. Google Search Console に登録
3. Xシェア文言とサイトURLを確認
4. アフィリエイトリンクを正式な広告リンクに差し替え
5. 広告・PR表記を必要に応じて追加
