TRUST & POLICY / 04

お問い合わせ

掲載内容の誤りや修正のご連絡、サイトに関するお問い合わせはGitHub Issuesで受け付けています。

お問い合わせ方法

下記のGitHub Issuesページを開き、「New issue」から内容を投稿してください。GitHubへのログインが必要になる場合があります。

GitHub Issuesを開く →

投稿前に必ずご確認ください

GitHub Issuesへの投稿内容は、インターネット上で公開されます。
氏名、住所、電話番号、メールアドレス、学校名・勤務先、アカウント情報など、個人を特定できる情報や公開を望まない情報は書かないでください。

お問い合わせ内容は、問題を確認できる範囲で具体的に記載してください。ただし、資格試験の個別の合否判定や、就職・収入に関する個別相談への回答を保証するものではありません。

ご連絡いただける内容

  • 掲載している資格情報の誤りや更新情報
  • 診断ページの表示・操作上の不具合
  • 広告表示や各ポリシーに関するご指摘
  • その他、サイト運営に関するご意見
